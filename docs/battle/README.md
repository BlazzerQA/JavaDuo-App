# Система Битв

## Обзор

Система битв позволяет игроку сражаться с противниками, используя нанятых юнитов. Битва происходит в реальном времени с поочередными ходами.

## Компоненты

### 1. BattleViewModel (`app/src/main/java/com/javadu/viewmodel/BattleViewModel.kt`)

Управляет состоянием битвы и логикой ходов.

#### Состояние (BattleState)
| Поле | Описание |
|------|----------|
| `playerArmy` | Список юнитов игрока |
| `enemyArmy` | Список юнитов противника |
| `battleLog` | История событий битвы |
| `isPlayerTurn` | Флаг хода игрока |
| `selectedUnitId` | ID выбранного юнита для атаки |
| `battleResult` | Результат битвы (победа/поражение) |
| `isLoading` | Флаг загрузки |
| `attackedPlayerUnitId` | ID юнита, который получает урон |

#### Методы
| Метод | Описание |
|-------|----------|
| `startBattle()` | Инициализация битвы, создание армий |
| `selectUnit(unitId)` | Выбор юнита игрока для атаки |
| `attackEnemy(targetUnitId)` | Атака выбранного врага |
| `enemyTurn()` | Ход противника (автоматически) |
| `checkBattleEnd()` | Проверка окончания битвы |
| `endBattle()` | Завершение битвы, сброс состояния |
| `retryBattle()` | Повторная попытка битвы |

### 2. UnitInBattle (`app/src/main/java/com/javadu/data/battle/UnitInBattle.kt`)

Модель юнита в битве.

#### Поля
| Поле | Описание |
|------|----------|
| `id` | Уникальный идентификатор |
| `icon` | Эмодзи/иконка юнита |
| `name` | Название юнита |
| `attack` | Атака |
| `defense` | Защита (броня) |
| `maxHp` | Максимальное здоровье |
| `currentHp` | Текущее здоровье |
| `isPlayer` | Принадлежность (игрок/враг) |
| `level` | Уровень юнита |

#### Вычисляемые свойства
| Свойство | Описание |
|----------|----------|
| `isAlive` | `currentHp > 0` |
| `hpPercentage` | Процент здоровья (0.0 - 1.0) |

### 3. BattleScreen (`app/src/main/java/com/javadu/ui/screens/BattleScreen.kt`)

UI экран битвы.

#### Элементы UI
- **Тулбар**: Название "Битва", кнопка "Назад", кнопка "Сбежать"
- **Лог битвы**: Скроллируемый список событий (80dp высоты)
- **Поле битвы**: Центрированные карточки юнитов
  - Верхний ряд: Враги
  - Разделительная линия
  - Нижний ряд: Игрок

### 4. Battlefield (`app/src/main/java/com/javadu/ui/components/Battlefield.kt`)

Компонент поля битвы с рядами юнитов.

### 5. BattleCard (`app/src/main/java/com/javadu/ui/components/BattleCard.kt`)

Карточка юнита с анимациями.

#### Элементы карточки
- **Иконка**: Центр
- **Название**: Под иконкой
- **Уровень**: Левый верхний угол (если > 1)
- **Броня**: Левый верхний угол
- **Здоровье**: Правый верхний угол
- **Атака**: Левый нижний угол

#### Анимации
| Анимация | Триггер | Описание |
|----------|---------|----------|
| Attack | Атака юнитом | Наклон и возврат |
| Damage | Получение урона | Тряска (3 раза) |
| TakeDamage | Атака по юниту | Красная рамка и фон |
| Death | Смерть юнита | Исчезновение (fade) |

## Логика битвы

### 1. Начало битвы
```
1. Загрузка нанятых юнитов из репозитория
2. Создание армии игрока (createPlayerUnit)
3. Создание армии противника (шаблон)
4. Инициализация состояния
```

### 2. Ход игрока
```
1. Игрок выбирает своего юнита (selectUnit)
2. Игрок кликает на врага (attackEnemy)
3. Расчёт урона: max(атака - защита, 1)
4. Обновление HP цели
5. Удаление убитых юнитов
6. Проверка окончания битвы
7. Передача хода противнику
```

### 3. Ход противника
```
1. Задержка 500мс
2. Выбор самого слабого врага (по HP)
3. Выбор самого слабого игрока (по HP)
4. Расчёт урона
5. Обновление HP цели
6. Удаление убитых юнитов
7. Проверка окончания битвы
8. Возврат хода игроку
```

### 4. Окончание битвы
| Условие | Результат |
|---------|-----------|
| Все игроки мертвы, все враги живы | Поражение |
| Все игроки живы, все враги мертвы | Победа |
| Все игроки мертвы, все враги мертвы | Победа (ничья) |

## Расчёт урона

```kotlin
fun calculateDamage(attackerAttack: Int, defenderDefense: Int): Int {
    val rawDamage = attackerAttack - defenderDefense
    return maxOf(rawDamage, 1)
}
```

Урон всегда минимум 1, даже если защита >= атаки.

## Уровни юнитов

При создании юнита в битве:
```
HP = baseHp + (level - 1) * 5
Attack = baseAttack + (level - 1) * 2
Defense = baseDefense + (level - 1) * 1
```
